# Glossary

- Ham: legitimate email.
- Spam: unwanted or malicious email.
- MBOX: a file format that stores email messages in one file.
- TF-IDF: a text feature method that scores words by importance.
- GridSearchCV: scikit-learn tool for trying parameter combinations.
- Vectorizer: the object that turns text into numbers.
- Confidence: a displayed score that estimates how strongly the model favors a class.
- Artifact: a saved file produced by training, such as a model pickle or vectorizer.
- Weighted F1: a metric that balances precision and recall while accounting for class frequency.
- Pickle: Python object serialization; convenient but unsafe for untrusted files.
